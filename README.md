Iddea general de esto:
Generar un turn-o-matic que trabaje de manera local y respalde en la nube.
Este debe trabajar sobre un servidor local (totem de consulta).
Para los escritorios de atención, se debe realizar una de las siguientes alternativas:
a.- Un pequeño programa para liberar un turno.
b.- Un acceso al totem por web para liberar el turno.
